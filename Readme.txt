Инструкция

Программа предполагает использование консольного интерфейса.
Каждое действие совершается в игре путём ввода соответствующей команды.

(Используемая версия java - java 15)
_____________________________

Команды для расстановки кораблей:
_____

set <x> <y> <Палубность> <Ориентация>

ПРИМЕР: set 0 0 2 true

Устанавливает корабль по заданным координатам 
(значения координат в диапазоне 0 - 9, x считается слева направо, у - сверху вниз)
с заданной палубностью (1 - 4) и ориентацией (указать true для горизонтальной ориентации, false - для вертикальной,
для однопалубного корабля - любое значение). Для многопалубных кораблей указываются координаты верхней левой клетки.

_____

del <x> <y>

ПРИМЕР: del 0 1

Удаляет корабль. Для удаления неправильно установленного корабля 
указываются координаты ЛЮБОЙ из клеток, которые он занимает. Удаляется весь корабль целиком.
_____

hide

Отключает показ поля на фазе установки кораблей. Отменить действие невозможно. 
Является служебной комадой, использующейся для тестирования. В обычной игре применять не рекомендуется.
_____

Пока не реализованные команды:

mine <x> <y>
Устанавливает мину

ms <x> <y>
Устанавливает минный тральщик

______________________________
Атака кораблей.

Для атаки соперника во время фазы боя нужно просто ввести в консоль через пробел
координаты атакуемой клетки, без каких-либо других пояснений.

ПРИМЕР: 3 5
